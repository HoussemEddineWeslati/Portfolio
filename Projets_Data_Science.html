OpenAI Function Calling

Ce projet est un système de questions-réponses alimenté par GPT-3 d'OpenAI et intégré à une base de données MySQL. Il offre une interface conviviale pour interroger les données des étudiants, telles que les notes et les frais. Les utilisateurs saisissent des questions, et le système fournit des réponses contextuelles, en faisant un outil précieux pour les établissements éducatifs et les applications axées sur les données.
